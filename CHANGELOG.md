# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-11-21

### Added
- Comprehensive `pyproject.toml` configuration with full metadata
- Support for Python 3.8 through 3.12
- Automated CI/CD workflows (tests.yml, publish.yml)
- Test suite with basic CLI tests
- Code quality checks: Black, Ruff, MyPy
- Coverage reporting integration with Codecov
- GitHub Actions trusted publisher setup for PyPI
- Extended project metadata and classifiers

### Changed
- Updated version to 0.2.0
- Enhanced email metadata (users.noreply.github.com)
- Improved project keywords and description
- Optimized GitHub workflows with test matrix

### Fixed
- PyPI trusted publisher configuration
- Build system dependencies

## [0.1.0] - 2025-11-21

### Added
- Initial project structure
- Basic CLI module with Gemini integration
- Initial pyproject.toml
- GitHub workflows setup
